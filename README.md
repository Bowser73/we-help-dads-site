## Domain vs Website Files

In simple terms:

- GoDaddy manages the domain name.
- GitHub stores the website files and code.
- GitHub Pages or another hosting provider publishes the website online.
- The domain points visitors to the hosted website.
- Jeff can own the domain while Nate maintains the website files as a collaborator.
